Principles of Web Design

  1. Colour Theroy

    • RED : Love, Energy, Intensity
    • YELLOW : joy, intellect, attention
    • GREEN : freshness, safety, growth
    • BLUE : stability, trust, serenity
    • PURPLE : roualty, wealth, femininity.
    1. Combination of colours.
      1. Analogous Colours : Taking two colours right next to each others & these good for nav bar or body or logo & background.
      2. Complementary Colours : Extact opposite to each other for POP but donot try on text background. keep it to logo , icons
      3. traidic or square : select diff 3 or 4 colours from colour pallet.

    Get a reference from these site's. Colour-hunt. OR adobe-color.

  2. Typography

    1. serif : Traditional(mini pro), STABLE(Trajan), RESPECTABLE(Baskerille) [serious + older]
    2. sans-serif : Sensible(Helvetica), SIMPLE(Avenir), STRAOGTHFORWARD(Din)
    3. Script : Personal(freestyle script), creative(adios script pro), elegant(snell roundhand)
    4. display : friendly(vag rounded), loud(Gin), amusing(thirsty rough)
    5. modern : stylish(sackers gothic), chic(gotham), smart(futura)

  3. User Interface Design

    1. Hierarchy : {mixture of BIG(+) & small words(-)}. Heighlight main words with big fornt & remains small. Colour {Primary & secondary/mix colour as your main references}.
    2. Layout : 40/60 characters per line.
    3. Alignment : all in one side.
    4. white space : adding space between each elements on screen.
    5. Audience : diffrentiate category like for childern, adults or event like party, office, frunral.
