Font size using px, point, em, rem.
  1. Using Pixel "1 pixel is 1/96 0f an inch
  2. Using Point "1 point is 1/72 of an Inch"
  3. Using Em "1 em is 100% the size of the parent element
  4. Using Rem "1 rem is 100% the root element {root element is html}

Font family declaration. Inside CSS file we can declare font family.
  • For single word font family, & generic
    .h1{
    font-family: Helvetica, sans-serif;
    }
  • For many word of fornt family, & generic
    .h1{
    font-family: "Times New Roman", serif;
    }
