Media Quries

Meadia Quries used to set the max width for the web page, webpage adjust as per the max width limit.
@media (max-width: 600px) {
"Css for screens below or equal to 600px wide"
}

Also, we can set min-width for the screen size like "@media(min-width: 300px)" that means the CSS will be apply to 300px+ size screen.
ex. @media (min-width:400px) and (max-width:600px){
