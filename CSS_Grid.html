CSS Grid

Here we use "display-grid" property. is used to create two dimentional 2D layout.
Column and Row.
Container contains that contains all items this is usually a div its called "grid container".
The items that are place inside the grid are "grid items".
we cretaes the rows & columns in grid these are called as "Tracks" row tracks & column tracks. We resize these tracks using "grid-template-columns/rows".
smallest unit in grid is "grid cell" but can be diff in size but set in row and column.
grid lines are vertical & horizontal lines. we can controll them using "gap" property for space.

  • we can use "grid-template: 50px 100px / 100px 200px" 1st is rows & 2nd is columns instead of "grid-template-columns: 350px 200px ; this, but it is not responsive because we gave fix px size.
  • we can use "grid-template-rows: 100px auto; grid-template-columns: 200px auto ;" it will give us some responsiveness for 2nd column 100% width becus it take hole width & 2nd row for the content quantity. Also we can use "grid-template: 100px auto / 200px auto"
  • we can use "grid-template-rows: repeat(2, 200px); "grid-template-columns: repeat(3, 100px) it creates 2 rows & 3 columns. used for reduce the repeat frame/px function
  • for arraning grid items in cells we can use "grid-column-start: 1; grid-column-end: 3; grid-row-start: 2; grid-row-end: 3; for this, we can use the replacement "grid-area: 2 / 1 / 3 / 3" the sequence is (grid-row-start) / (grid-column-start) / (grid-row-end) / (grid-column-end)
