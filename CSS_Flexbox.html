Flex box

Here we use display-flex property.
CSS flexbox is used to create complex 1D layout on the webpage i.e. column/row, it does the intraction with user.

  • We can define flex using"display:flex;
  • This is the default direction of flex box "flex-direction: row;"left to right and we can add space between two div/p using "gap:10px;" or two columns /row.
  • We can set height of the layout using "flex-basis: 100px;" to all child elements. using parent > *
  • we can declare div + p element in inline as they are block leve elements flexbox allows us to show the content as inline "display: inline-flex;".
  • We can use order property to arrange the child element in order as we want to see them, order default goes to left to right. Greater the oder arrangnment is later/ last.
  • we can use "flex-wrap: wrap to wrap the content if there is not much space available for that content, and it applies on the container/parent.
  • we can use this for horizontal position"justify-contet: flex-start/end/center & space-between/space-evenly/space-around;" on parent to arrenge content left/right/center.
  • we can arrange the content vertically using "align-items: flex-start/center/flex-end/strech/baseline;" but for this we have to give the height to parent"height:70vh;"for this property we have to set "flex-wrap:nowrap"
  • "align-self: flex-start; property is only for child applicable
  • "align-content" is only work if "wrap"
  • max-width is show how long the content strech & min-width shows how small the content shrink.
  • In shrink, by default flex occupy the whole width of row.
    Content width < width < flex-basis < min-width/max-width.
    If we set flex-basis:200px; & max-width:100px(max wide range is 100px); then the width length is 100px(width < flex-basis) & width of flex-basis content is max 200px will ignore (not occupy the whole width of page.)
    max-width is full length of sentence. if max-width value is smaller than basis then max-width is countable but if max-width value is greater than basis then basis vaule is countable. min-width is full length of longest word. if min-width is greater than basis then min-width is countable but it will not goint to shrink less than min-width value.
